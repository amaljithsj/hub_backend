# Trace Backend Features

Analyze backend feature implementations.

## Focus Areas

- Models
- Services
- Routes
- Schemas
- Dependencies

## Goals

- Locate feature ownership
- Trace implementation flow
- Identify related backend assets

## Output

Return:
- Matching files
- Related services
- Feature locations

## Workflow

1. Receive feature name
2. Search backend source tree
3. Locate matching files
4. Return findings
